A flask web app

Shows a web page with a form.

Upon submission both javascript and python checks if the user has inputted a name.
if no name is inputted, javascript will prompt for a name or if javascript is disabled, the user will be redirected to an error screen prompting for a name

if Name is provided, the name will be added to a csv file
and the user will be redirected to a web page showing all the names inside the csv file.
